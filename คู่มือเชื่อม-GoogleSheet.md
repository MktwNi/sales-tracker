# คู่มือเชื่อมแอปกับ Google Sheet (วิธี SheetDB — ไม่ต้องเขียนโค้ด)

แอปของคุณยังเป็น **หน้าเว็บเหมือนเดิมทุกอย่าง** คู่มือนี้แค่ทำให้หน้าเว็บส่งข้อมูลไปเก็บใน Google Sheet ได้
ใช้บริการชื่อ **SheetDB** เป็นตัวกลาง — สมัครเว็บเดียว ไม่ต้องเขียนโค้ดอะไรเลย ใช้เวลาประมาณ 5 นาที

---

## สิ่งที่ต้องมี
- บัญชี Google (Gmail) — mktw.zzz@gmail.com ที่ใช้อยู่
- ไฟล์แอป เปิดในเบราว์เซอร์ Chrome / Edge / Safari

---

## ขั้นที่ 1 — สร้าง Google Sheet และวางหัวคอลัมน์

1. เปิด **https://sheets.google.com** → คลิก **"ว่างเปล่า (Blank)"** สร้างไฟล์ใหม่
2. ตั้งชื่อไฟล์ก็ได้ เช่น `ฐานข้อมูลการขาย`
3. กลับมาที่แอป → กดปุ่ม **☁ Google Sheet** (มุมขวาบน) → กดปุ่ม **"📋 คัดลอกหัวคอลัมน์"**
4. กลับไปที่ Google Sheet → คลิกที่ **ช่อง A1** (ช่องซ้ายบนสุด) → วาง (Ctrl+V)

   วิธีนี้ดีสุด เพราะปุ่มจะก๊อปแบบ tab พอวางที่ A1 มันจะแตกเป็นช่องเองอัตโนมัติ ไม่ต้อง Split

   หัวคอลัมน์ชุดล่าสุดมี **21 คอลัมน์** (ขึ้นต้นด้วย `year`, มีคอลัมน์สถานะ และ `syncId`):

   `year, section, client, contactName, phone, email, resp, referral, contactDate, jobStatus, closedDate, statusJob, statusDeal, statusOverall, forecast, actual, source, service, progress, syncId, config`

   > ❗ ต้องวางที่ A1 เท่านั้น และอย่าแก้ชื่อหัวคอลัมน์ (ต้องตรงกับที่แอปส่งมา)
   >
   > 📌 **ถ้าเคยตั้ง Sheet ไว้ก่อนหน้านี้** (หัวคอลัมน์ชุดเก่า) ให้ลบข้อมูลใน Sheet ทิ้งทั้งหมดก่อน แล้ววางหัวคอลัมน์ชุดใหม่นี้ จากนั้นกด "บันทึกขึ้น Sheet" ใหม่อีกครั้ง

---

## ขั้นที่ 2 — สมัคร SheetDB และเชื่อม Sheet

1. เปิดแท็บใหม่ไปที่ **https://sheetdb.io**
2. กด **Sign up** / **Login** — เลือก **Continue with Google** แล้วล็อกอินด้วย Gmail ของคุณ (ง่ายสุด)
3. กดปุ่ม **Create new API**
4. มันจะขอ **ลิงก์ของ Google Sheet** — กลับไปที่ไฟล์ Sheet ของคุณ คัดลอก URL จากแถบที่อยู่ของเบราว์เซอร์ (เริ่มด้วย `https://docs.google.com/spreadsheets/d/...`)
   - ครั้งแรกอาจต้องกด **อนุญาต (Authorize / Allow)** ให้ SheetDB เข้าถึง Google Sheet ของคุณ — กดยอมรับ
5. วางลิงก์ Sheet ลงไป → กด **Create API**

---

## ขั้นที่ 3 — คัดลอก API URL มาใส่ในแอป

1. เมื่อสร้างเสร็จ SheetDB จะแสดง **API URL** หน้าตาแบบนี้:

   ```
   https://sheetdb.io/api/v1/xxxxxxxxxxxxx
   ```

2. กดปุ่ม **Copy** ตรงนั้นเพื่อคัดลอก
3. กลับมาที่แอป → ในหน้าต่าง **☁ Google Sheet** → วาง URL ในช่อง **"SheetDB API URL"**
4. กดปุ่ม **"☁ บันทึกขึ้น Sheet"** หนึ่งครั้ง
5. กลับไปดูที่ Google Sheet → ข้อมูลลูกค้าทั้งหมดจะโผล่ขึ้นมาในตาราง ✅

**เสร็จแล้ว!** ข้อมูลของคุณอยู่ใน Google Sheet เรียบร้อย

---

## ใช้งานประจำวัน

| ต้องการ | ทำอย่างไร |
|---|---|
| ส่งข้อมูลล่าสุดขึ้น Sheet | กด ☁ Google Sheet → **"☁ บันทึกขึ้น Sheet"** |
| ดึงข้อมูลจาก Sheet มาที่เครื่อง | กด ☁ Google Sheet → **"⬇ ดึงจาก Sheet"** |
| ให้ส่งขึ้นเองอัตโนมัติ | ติ๊กช่อง **"ซิงค์อัตโนมัติทุกครั้งที่แก้ข้อมูล"** |

### เรื่องซิงค์อัตโนมัติ (อ่านสักนิด)
SheetDB **แพ็กฟรีมีลิมิตจำนวนครั้งต่อเดือน** (ประมาณหลักร้อยครั้ง) ถ้าเปิดซิงค์อัตโนมัติแล้วแก้ข้อมูลถี่ ๆ อาจใช้โควตาหมดเร็ว

**คำแนะนำ:**
- ถ้าแก้ไม่บ่อย → เปิดซิงค์อัตโนมัติได้สบาย ๆ
- ถ้าแก้เยอะ ๆ ทั้งวัน → ปิดอัตโนมัติ แล้วกด **"บันทึกขึ้น Sheet"** เองวันละครั้งสองครั้งพอ

---

## ใช้หลายเครื่อง / หลายคน
- เครื่องอื่นใส่ **API URL เดียวกัน** แล้วกด **"⬇ ดึงจาก Sheet"** เพื่อโหลดข้อมูลล่าสุด
- การบันทึกเป็นแบบ "เขียนทับทั้งหมด" ถ้า 2 คนแก้พร้อมกัน คนบันทึกทีหลังจะทับ — แนะนำตกลงกันว่าใครแก้ช่วงไหน หรือก่อนแก้ให้ "ดึงจาก Sheet" ก่อนทุกครั้ง

---

## แก้ปัญหาที่พบบ่อย

**กดบันทึกแล้วขึ้น error / ไม่เข้า Sheet**
- ตรวจว่า URL เป็นของ SheetDB และคัดลอกครบ (`https://sheetdb.io/api/v1/...`)
- ตรวจว่าหัวคอลัมน์ใน Sheet อยู่ที่แถวแรก (A1) และสะกดตรงกับที่คัดลอกมา

**ข้อมูลใน Sheet ไม่อัปเดต**
- กด **"บันทึกขึ้น Sheet"** อีกครั้ง แล้วรีเฟรชหน้า Google Sheet

**อยากเปลี่ยน Sheet ปลายทาง**
- สร้าง API ใหม่ใน SheetDB กับ Sheet ใหม่ แล้วเอา URL ใหม่มาวางแทน

---

## ข้อมูลหลายปี (อัปเดตแล้ว — ปลอดภัย)
ระบบใส่คอลัมน์ **year** ให้แล้ว ข้อมูลแต่ละปี (พ.ศ.) จะอยู่ใน Sheet เดียวกันได้โดย**ไม่ทับกัน** เวลากดบันทึก ระบบจะแทนที่เฉพาะแถวของปีที่เปิดอยู่เท่านั้น และเวลากด "ดึงจาก Sheet" จะดึงเฉพาะปีที่เลือก

> เคล็ดลับความปลอดภัย: ระบบจะ "เขียนข้อมูลชุดใหม่ก่อน แล้วค่อยลบชุดเก่า" ดังนั้นถ้าเน็ตหลุดกลางคัน ข้อมูลเดิมใน Sheet จะไม่หาย (อย่างมากแค่มีซ้ำชั่วคราว ระบบจะเลือกชุดล่าสุดให้เองตอนดึง)
