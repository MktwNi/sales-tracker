โน้ต
☁ เชื่อม Google Sheet (SheetDB)
วิธีตั้งค่า (ทำครั้งเดียว ~5 นาที ไม่ต้องเขียนโค้ด):
  1. สร้าง Google Sheet ใหม่ 1 ไฟล์ แล้ววางหัวคอลัมน์ที่ ช่อง A1 (ระบบจะแยกเป็นคอลัมน์ให้เอง):
  2. เข้า sheetdb.io → สมัคร/ล็อกอิน (ใช้ Google ได้) → กด Create new API → วางลิงก์ Google Sheet ของคุณ → กด Create
  3. จะได้ API URL หน้าตาแบบ https://sheetdb.io/api/v1/xxxxxxxx → คัดลอกมาวางด้านล่าง
  4. กด “☁ บันทึกขึ้น Sheet” ครั้งแรกเพื่อส่งข้อมูลขึ้นไป
หมายเหตุ: SheetDB แพ็กฟรีมีลิมิตจำนวนครั้ง/เดือน — ถ้าแก้ข้อมูลบ่อยมาก แนะนำปิดซิงค์อัตโนมัติแล้วกด “บันทึกขึ้น Sheet” เองเป็นช่วง ๆ
